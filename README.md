#### yii2 project for doc management

- 基于Yii2开发的文档管理系统
- 在线使用： https://ydc.jeen.wang

###### 概要功能描述

- 基于editor.md提供markdown富文本编辑，支持图片上传
- 文档项目成员权限分级，创建人、管理员、编辑、查阅
- 文档目录分层，目前支持3层(含文档根目录)
- 账号+微信多对多关联绑定
- 基于文档内容的简单搜索(不含标题及目录)
- 文档项目导出pdf，支持图片导出
- PDF支持按项目、目录、文档分级导出，项目及目录限创建人

###### 项目调整方向

- 简化文档中不必要的字段信息
